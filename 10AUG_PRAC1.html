WEB SITE

Colletion Of Web Pages With Realated Content Within Same Domain.

For Users Web Site Means Images , Color , Button , Video , Form , Text , Line , Box ETC In Organized Way. OR For Some People It Is Like For EX - www.amazon.com

For Web Server & Web Browser(Client Device Application) It Is HTML , CSS , JAVA SCRIPT & Back End Files (Depending Upon The Programming Language) And Data. It Is Set Of Different Styling , Scripting , Data Sending And Structring Files Which Are Linked Together.


WWW

World Wide Web , Network Of Servers Over The Internet Which Allows Web Resource To Be Accessed Over The Internet.


HTTP / HTTPS

Hyper Text Transfer Protocol Is For Transmitting Hypermedia Documents Such As HTML HTTP Is Responsible For Communication Of Web Server And Browser. S In HTTPS Stands For Secure Connection, Simply The Lock Symbol On The Left Side Of WWW.


CLIENT SERVER

A Client Is A Person Who Sends A Request Of Website To The Server With The Help Of Browser In His/Her Device.
Server Is Computer That Host A Website On The Internet And Give Back Response To Cilents.


DOMAIN NAME 
        Domain Name Is Web Address That Is linked To An IP Address
        
      EX - https://www.amazon.com
      
      HERE :
      https: Is Protocol
      www Is Sub Domain Which Will Be Common In Every Website
      amazon Is Second Level Domain Or Domain Name
      com Is Top Level Domain
      
      Second Level Domain + Top Level Domain = Root Domain (Which U Can Register)
      
      Top Level Domain Name Can Be diffrent According To The Geographical Location And Oraganizationn type
